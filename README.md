Kris Torres’s Website
=====================

[![React](https://img.shields.io/badge/React-16-blue.svg?style=flat&maxAge=2592000)](https://reactjs.org)

This repository contains the code for my upcoming developer portfolio. The
website will be written in JSX (a JavaScript extension that allows HTML
generation), using React.

Quick Overview
--------------

```sh
git clone https://github.com/kristorres/kristorres.github.io.git
cd kristorres.github.io
npm install
npm run build-dev
```

Now go to _build/dev/index.html_ on your favorite browser to view the web app. 🙂
