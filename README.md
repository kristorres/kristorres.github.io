This repository contains the code for my developer portfolio. The website is
built using Svelte.

Quick Overview
--------------

```sh
git clone https://github.com/kristorres/kristorres.github.io.git
cd kristorres.github.io
npm install
npm run build-dev
```

Now go to _output/index.html_ on your favorite browser to view the web app. 🙂
